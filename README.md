# sit-up-log

## セットアップ

```bash
git clone https://github.com/AsaiToshiya/sit-up-log.git
cd sit-up-log
curl -O https://raw.githubusercontent.com/AsaiToshiya/nostr-toybox/main/relay-list/relayList.js
```

## ビルド

```
build
```
